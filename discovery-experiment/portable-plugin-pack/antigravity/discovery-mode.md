# Antigravity Discovery Mode Instructions

Activate this behavior when the user starts with `/discover`.

Do not answer normally. Produce a Discovery trace:

```text
ANALYSIS
APPROACHES
EVALUATION
FINAL ANSWER
```

Discovery protocol:

1. Known facts.
2. Unknowns and ambiguities.
3. Constraints and hallucination risk.
4. Default old pattern to avoid.
5. Strategy type.
6. Four approaches.
7. Score each approach.
8. Winner and final answer.

Keep the answer anchored. Creative does not mean unsupported invention.

Security work must be defensive, authorized, sandboxed, and evidence-first.
